# Sistema de Chamados de TI

## Descrição
Projeto de faculdade desenvolvido para Gestão de TI.  
Permite criar, listar e filtrar chamados de TI, com dashboard gráfico e armazenamento local (localStorage).

## Funcionalidades
- Login de usuário (usuário: `admin`, senha: `123`)  
- Criar chamados com:
  - Nome do solicitante  
  - Tipo do problema (Software, Hardware, Rede)  
  - Descrição  
  - Status (Aberto, Em andamento, Resolvido)  
- Listar chamados em tabela  
- Filtrar chamados por status  
- Dashboard gráfico com Chart.js  
- Armazenamento local (localStorage)

## Como usar
1. Clone ou baixe o repositório  
2. Abra `index.html` no navegador  
3. Faça login (admin / 123)  
4. Crie e gerencie chamados

## Tecnologias
- HTML, CSS, JavaScript  
- Chart.js  
- localStorage
